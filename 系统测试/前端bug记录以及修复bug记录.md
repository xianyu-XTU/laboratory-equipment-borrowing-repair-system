# 前端bug记录以及修复bug记录

## 1. 权限菜单显示异常
**现象**：学生账号登录后显示管理员菜单，或管理员缺少功能菜单。
**原因**：前端菜单写死，未根据后端返回角色动态渲染。
**解决方法**：登录后获取用户角色，前端按角色过滤并动态渲染菜单。

## 2. 设备列表数据空白
**现象**：设备列表页面无数据展示，控制台无报错。
**原因**：前后端字段名不一致，或未正确解析接口返回列表。
**解决方法**：核对前后端字段，正确解析并赋值列表数据。

## 3. 借用申请状态不更新
**现象**：提交借用申请成功，设备状态仍显示“可借用”。
**原因**：提交后未刷新列表，或前后端状态码不匹配。
**解决方法**：申请成功后刷新列表，统一前后端状态码定义。
