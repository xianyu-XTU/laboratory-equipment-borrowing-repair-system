# 后端接口测试文档

## 一、用户模块
登录：输入账号密码登录，自动获取 Token，用于后续所有接口的身份认证  
用户列表：查看系统内所有用户信息  
新增用户：添加新的系统用户  
修改用户：编辑已有用户的资料信息  
删除用户：根据用户 ID 删除指定用户  
## 二、设备模块
设备列表：查看所有实验室设备信息  
新增设备：录入新设备信息至系统  
修改设备：编辑设备名称、运行状态、所属实验室等信息  
删除设备：根据设备 ID 删除指定设备  
设备详情：单独查看某一台设备的完整详细信息  
## 三、实验室模块
实验室列表：查看所有实验室基础信息  
新增实验室：添加新的实验室信息  
修改实验室：修改实验室名称、地址、管理员等信息  
删除实验室：根据实验室 ID 删除指定实验室  
## 四、设备借用模块
借用列表：查看所有设备借用记录  
申请借用：学生提交设备借用申请  
审批借用：管理员审核并通过借用申请  
设备归还：登记设备归还操作，更新状态为已归还  
## 五、设备报修模块  
报修列表：查看所有设备故障报修记录  
申请报修：提交设备故障报修申请单  
处理报修：管理员完成维修，更新报修状态
